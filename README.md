## Install

```bash
$ npm i -g httpdumper
```

## Usage

```bash
$ httpdumper --port="3000" --host="http://localhost:9000" --uplaodDir="./uploads/"
```

```bash
$ npx httpdumper --port="3000" --host="http://localhost:9000" --uplaodDir="./uploads/"
```
